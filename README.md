## vue商城管理后台 [前往vue3版本](https://github.com/shaoky/sky-vue3-mall-adminm)

> 项目采用vue-cli构建

### 技术栈

```
vue2 vue-router vuex es6 less Element UI lodash.js
```
### 安装

通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```

启动服务(http://localhost:8020)

```
npm run dev
```

发布代码
```
npm run build
```
